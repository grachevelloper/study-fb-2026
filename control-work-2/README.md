# Практические работы 7–12

Цикл практических заданий по разработке fullstack-приложения с использованием Node.js и React.js. В ходе работ реализованы: REST API, аутентификация с JWT-токенами, ролевая модель доступа, а также клиентская часть.

---

## Практическая работа 7 — REST API на Node.js

Создано серверное приложение на Node.js с базовыми маршрутами для управления пользователями и товарами.

**Маршруты**

| Метод | Маршрут | Назначение |
|-------|---------|------------|
| POST | /api/auth/register | Регистрация пользователя |
| POST | /api/auth/login | Вход в систему |
| POST | /api/products | Создание товара |
| GET | /api/products | Получение списка товаров |
| GET | /api/products/:id | Получение товара по идентификатору |
| PUT | /api/products/:id | Обновление товара |
| DELETE | /api/products/:id | Удаление товара |

**Модели данных**

Пользователь:
- идентификатор
- электронная почта
- имя
- фамилия
- пароль (хешируется, электронная почта используется как логин)

Товар:
- идентификатор
- наименование
- категория
- описание
- цена

## Практическая работа 9 — Refresh-токены

В приложение из работы 8 добавлена генерация refresh-токенов и маршрут для их обновления.

**Добавленные маршруты**

| Метод | Маршрут | Назначение |
|-------|---------|------------|
| POST | /api/auth/refresh | Обновление пары токенов |

**Формат ответа при входе**

```json
{
  "accessToken": "...",
  "refreshToken": "..."
}
```

Маршрут обновления принимает refresh-токен и возвращает новую пару в том же формате.

## Практическая работа 11 — Ролевая модель доступа

В приложение из работы 10 добавлена система ролей с разграничением прав.

**Роли пользователей**

| Роль | Описание |
|------|----------|
| Гость | Неаутентифицированный пользователь |
| Пользователь | Аутентифицированный пользователь (только просмотр) |
| Продавец | Сотрудник (добавление и редактирование товаров) |
| Администратор | Полные права + управление пользователями |

**Права доступа к маршрутам**

| Метод | Маршрут | Доступ | Назначение |
|-------|---------|--------|------------|
| POST | /api/auth/register | Гость | Регистрация |
| POST | /api/auth/login | Гость | Вход |
| POST | /api/auth/refresh | Гость | Обновление токенов |
| GET | /api/auth/me | Пользователь | Текущий пользователь |
| GET | /api/users | Администратор | Список пользователей |
| GET | /api/users/:id | Администратор | Пользователь по идентификатору |
| PUT | /api/users/:id | Администратор | Обновление пользователя |
| DELETE | /api/users/:id | Администратор | Блокировка пользователя |
| POST | /api/products | Продавец | Создание товара |
| GET | /api/products | Пользователь | Список товаров |
| GET | /api/products/:id | Пользователь | Товар по идентификатору |
| PUT | /api/products/:id | Продавец | Обновление товара |
| DELETE | /api/products/:id | Администратор | Удаление товара |

**Расширенная функциональность клиента**

Управление товарами:
- просмотр списка
- просмотр деталей
- создание
- редактирование
- удаление

Управление пользователями (только для администратора):
- просмотр списка пользователей
- редактирование информации
- блокировка и разблокировка
