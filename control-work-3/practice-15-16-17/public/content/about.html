О приложении

Версия 1.0.0

Учебный TODO-менеджер с напоминаниями, реализованный в рамках практических занятий 15, 16 и 17 дисциплины «Фронтенд и бэкенд разработка».

Реализованные возможности:

  • Progressive Web App (PWA) с поддержкой установки
  • HTTPS через mkcert (локальный доверенный сертификат)
  • App Shell архитектура — мгновенная загрузка каркаса
  • Service Worker: Cache First (статика) + Network First (контент)
  • Офлайн-режим — задачи доступны без сети
  • WebSocket (Socket.IO) — уведомления в реальном времени
  • Push-уведомления через Web Push API + VAPID
  • Напоминания с планированием на стороне сервера (setTimeout)
  • Кнопка «Отложить на 5 минут» в push-уведомлении
  • Хранение данных в localStorage
  • TypeScript на сервере и клиенте
